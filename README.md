# ecommerce-manual-testing-project
Manual QA test cases for a demo E-Commerce web application demonstrating functional, positive, and negative testing.
-Commerce Manual Testing Project

This repository contains a set of manually designed test cases for a demo E-Commerce website. 

## Scope:
- User Registration
- User Login
- Add to Cart functionality
- Product Sorting (Price & Name)
- Brand item count validation

## Test Types:
- Positive Testing
- Negative Testing
- Functional Testing

## Test Case Structure:
Each test case includes:
- Test ID
- Test Title
- Preconditions
- Steps
- Expected Results
- Priority

## Tools:
- Manual Test Case Design
- Google Sheets / Excel
